## ‚ú® Features

### Commands (8)
1. **Connect Device** - Connect to Android device
2. **Quick Actions** - Show all actions menu
3. **Connect to IP** - Connect to specific IP
4. **Screenshot** - Take screenshot
5. **List Devices** - Show all devices
6. **Reset Connection** - Reset saved connection
7. **Connect with Profile** - Use performance profile
8. **Install Dependencies** - Install scrcpy & ADB

### Keyboard Shortcuts
- `Ctrl+Alt+S` (Mac: `Cmd+Alt+S`) - Quick Actions Menu
- `Ctrl+Alt+C` (Mac: `Cmd+Alt+C`) - Connect Device
- `Ctrl+Alt+P` (Mac: `Cmd+Alt+P`) - Take Screenshot

### Settings
- **Bitrate**: Video bitrate (default: 8M)
- **Max Size**: Maximum resolution (default: 1024)
- **Max FPS**: Maximum frames per second (default: 60)
