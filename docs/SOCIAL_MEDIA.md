# üöÄ Scrcpy Smart Connect: Never Use USB Cables for Android Development Again

Are you tired of being tethered to your Android device during development? Let me introduce you to **Scrcpy Smart Connect** - a smart wireless connection manager that makes cable-free development a breeze.

## The Problem

As mobile developers, we constantly need to:
- Test apps on real devices
- Use hot reload during development
- Record screens for demos
- Debug issues that only happen on physical devices

But USB cables are:
- ‚ùå Restrictive
- ‚ùå Easy to forget

## The Solution

Scrcpy Smart Connect automatically manages wireless connections to your Android devices using scrcpy.

### Key Features

‚úÖ **Auto-reconnect**: Remembers your device IP
‚úÖ **USB Fallback**: Seamlessly falls back if wireless fails
‚úÖ **Cross-platform**: Linux, macOS, Windows
‚úÖ **One-line install**: Get started in seconds
‚úÖ **Multiple devices**: Manage several devices easily

## Quick Start

```bash
# Install
curl -fsSL https://raw.githubusercontent.com/tareq-alomari/scrcpy-smart/main/install.sh | bash

# Connect (first time with USB)
phone

# Disconnect USB - it's now wireless!
```

## How It Works

1. **First connection**: Plug in USB, run `phone`
2. **Script magic**: Detects device, gets IP, switches to wireless
3. **Future connections**: Just run `phone` - no USB needed!
